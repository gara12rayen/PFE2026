# 3LM Solutions — Frontend combiné (ATS + Formation)

## Démarrage

```bash
npm install
npm start    # http://localhost:3000
```

## Navigation
- Page d'accueil : sélecteur entre Recrutement et Formation
- Espace Recrutement : ATS complet (candidat / RH / admin)
- Espace Formation : agent formateur IA (employé / admin formation)
- Bouton "← Accueil" en bas à droite pour revenir au sélecteur
